# Calculadora de Consumo de Energia Elétrica

Projeto em Python para estimar o consumo mensal de energia elétrica de aparelhos domésticos, com base na potência e no tempo de uso diário.

# Descrição
O programa solicita ao usuário o nome do aparelho, sua potência em watts e quantas horas por dia ele fica ligado. A partir dessas informações, calcula o consumo estimado em kWh por mês.

# Exemplo de uso
Qual o nome do aparelho? televisão
Qual a potência do aparelho? 150
Quantas horas por dia o aparelho fica ligado? 12

Aparelho: televisão
Consumo estimado: 54.0 kWh/mês

# Fórmula do cálculo
Consumo (kWh/mês) = Potência (W) × Horas/dia × 30 ÷ 1000
